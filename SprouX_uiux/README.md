# SprouX Design System

Component library powered by Shadcn + Tailwind CSS + React + TypeScript.

## Tech Stack

- **Vite** — Build tool
- **React 19** — UI framework
- **TypeScript** — Type safety
- **Tailwind CSS v4** — Utility-first CSS
- **Shadcn** — Component foundation (new-york style)
- **CVA** — Component variant management

## Getting Started

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
